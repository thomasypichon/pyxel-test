## Auto-Save Setting
To set up auto-save in Cursor:
1. Go to Cursor Settings (Cmd + ,)
2. Search for "auto save"
3. Change "Auto Save" to "afterDelay"
4. Set "Auto Save Delay" to 1000 (that's 1 second)
